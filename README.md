## 📌 Sobre

O **Hub Brain** é um app de produtividade escolar feito por um estudante, pra estudantes. Nasceu de uma necessidade real — um lugar único pra organizar notas, tarefas, horários e focar nos estudos, sem precisar ficar abrindo mil apps diferentes.

Tudo gratuito, tudo no navegador, funciona como PWA instalável no celular.

---

## ✨ Funcionalidades

| Feature | Descrição |
|---|---|
| 📊 **Notas** | Cadastre matérias e acompanhe seu desempenho por bimestre |
| 📋 **Agenda** | Organize tarefas e provas com alertas de urgência |
| ⏱️ **Modo Foco** | Timer flip clock com modo livre, histórico e XP |
| 🕐 **Horário** | Grade de horários semanal personalizada |
| 🏆 **Ranking** | Sistema de XP e patentes com ranking global |
| 🧠 **Estudos & IA** | Ferramentas de estudo com inteligência artificial |
| 📓 **Caderno** | Notas com checklist, anexos, foto e IA integrada |
| 📅 **Cronograma** | Planejamento inteligente de estudos com integração ao Foco |
| 🎯 **Conquistas** | Sistema de conquistas desbloqueáveis por ação |
| 🔔 **Notificações** | Push notifications via OneSignal |

---

## 🛠️ Stack

- **Frontend:** HTML5 · CSS3 · JavaScript Vanilla
- **Backend:** Firebase Firestore + Firebase Auth
- **Hospedagem:** Netlify
- **PWA:** Service Worker + Web App Manifest
- **Notificações:** OneSignal
- **Ícones:** Lucide Icons

## 🚀 Rodando localmente

```bash
# Clone o repositório
git clone https://github.com/wtfditoh/dtschool.git

# Abra o index.html no navegador
# ou use um servidor local como Live Server (VSCode)
```

> Não precisa de build. É HTML puro — abre e funciona.

## 📱 PWA

O Hub Brain funciona como Progressive Web App. Para instalar:

1. Acesse [hubbrain.netlify.app](https://hubbrain.netlify.app)
2. No navegador, toque em **"Adicionar à tela inicial"**
3. Use como um app nativo
